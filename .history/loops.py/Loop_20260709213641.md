what is loop ?

 Types of loops?
  1.for loop.
  2.while loop.
  

FOR LOOP:
   for loop using while we know how many time the loop what it run.
for variable name in  range(Stating,Ending,step):
  Statement


  eg:
  for i in range(0,10,1):
     print(i)  



WHILE LOOP:
   we donot know how many time loop want to run , at the time we use while loop.

eg :
i=1;
while i<=5:
   print(i)
   i=i+1


Break: 
  To stop the loop immediately .
  for i in range(1,6):
    if i==3:
      break
    print(i)

output:
1 2    

Continue:
 To skip the current loop.
 for i in range(1,6):
   if i==3:
     Continue
   print(i)  

output:
1 2 4 5
